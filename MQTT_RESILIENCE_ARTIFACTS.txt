QUICK START
================================================================================

1. Import components:
   from mqtt_resilience import CircuitBreaker, MQTTHealthMonitor
   from mqtt_resilience_example import MQTTResilienceManager

2. Load configuration:
   config = load_mqtt_config()

3. Initialize:
   cb = CircuitBreaker(config["circuit_breaker_failure_threshold"], ...)
   monitor = MQTTHealthMonitor(...)

4. Use:
   await cb.call(mqtt_operation)
   await monitor.record_subscriber_message(queue_depth=5)
   
5. Expose endpoints:
   handler = HealthCheckHandler(monitor, cb, start_time)
   health = await handler.handle_health_check()
   metrics = await handler.handle_metrics()
